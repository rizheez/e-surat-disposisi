# Panduan Menu Dasbor

## Fungsi Menu

Menu Dasbor adalah halaman awal setelah pengguna masuk ke aplikasi. Halaman ini digunakan untuk melihat ringkasan aktivitas persuratan dan menjadi titik awal sebelum membuka menu lain.

## Cara Menggunakan

1. Login ke aplikasi E-Surat.
2. Setelah berhasil login, pengguna akan diarahkan ke halaman Dasbor.
3. Perhatikan informasi ringkasan yang tersedia di halaman.
4. Gunakan sidebar kiri untuk berpindah ke menu lain seperti Surat Masuk, Surat Keluar, Disposisi, atau Master Data.

## Notifikasi

1. Klik ikon notifikasi di bagian kanan atas.
2. Sistem akan menampilkan daftar notifikasi yang tersimpan.
3. Jika belum ada notifikasi, panel akan menampilkan pesan bahwa tidak ada notifikasi.
4. Jika pengguna baru saja menerima tugas tetapi notifikasi belum terlihat, refresh halaman terlebih dahulu.

## Catatan Penggunaan

- Menu yang tampil di sidebar dapat berbeda antar pengguna, tergantung role dan hak akses.
- Jika ada menu penting yang tidak tampil, hubungi admin untuk pengecekan role.
- Dasbor tidak digunakan untuk menginput surat secara langsung. Input surat dilakukan melalui menu Surat Masuk, Surat Keluar, atau Generate Nomor sesuai kebutuhan.
