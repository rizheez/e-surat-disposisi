# Panduan Menu Arsip Surat

## Fungsi Menu

Menu Arsip Surat digunakan untuk melihat surat masuk dan surat keluar yang sudah diarsipkan. Menu ini membantu memisahkan surat aktif dari surat yang prosesnya sudah selesai.

## Jenis Arsip

Menu ini memiliki dua kategori:

1. **Arsip Surat Masuk**
2. **Arsip Surat Keluar**

## Melihat Arsip Surat Masuk

1. Buka menu **Arsip Surat**.
2. Pilih tab **Surat Masuk**.
3. Cari surat berdasarkan nomor, pengirim, perihal, atau tanggal jika tersedia.
4. Klik **View** untuk melihat detail surat.

## Melihat Arsip Surat Keluar

1. Buka menu **Arsip Surat**.
2. Pilih tab **Surat Keluar**.
3. Cari surat berdasarkan nomor, tujuan, perihal, atau tanggal jika tersedia.
4. Klik **View** untuk melihat detail surat.

## Mengembalikan Surat dari Arsip

1. Buka tab arsip sesuai jenis surat.
2. Pilih surat yang ingin dikembalikan.
3. Klik aksi **Unarchive** atau **Keluarkan dari Arsip**.
4. Surat akan kembali ke daftar surat aktif.

## Catatan Penggunaan

- Arsip sebaiknya digunakan untuk surat yang prosesnya sudah selesai.
- Mengarsipkan surat tidak sama dengan menghapus surat.
- Jika surat masih perlu ditindaklanjuti, jangan arsipkan terlebih dahulu.
