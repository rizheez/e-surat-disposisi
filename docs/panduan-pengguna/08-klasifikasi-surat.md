# Panduan Menu Klasifikasi Surat

## Fungsi Menu

Menu Klasifikasi Surat digunakan untuk mengatur kode dan jenis klasifikasi surat. Data ini dipakai saat membuat Surat Keluar dan Generate Nomor.

## Hak Akses

Menu ini dikelola oleh admin.

## Menambah Klasifikasi

1. Buka menu **Klasifikasi Surat**.
2. Klik **Buat**.
3. Isi **Kode** klasifikasi.
4. Isi **Nama** klasifikasi.
5. Pilih **Kategori**, misalnya internal, eksternal, atau khusus.
6. Isi **Kode Surat** jika klasifikasi memiliki format kode tertentu.
7. Isi **Keterangan** jika diperlukan.
8. Pastikan status **Aktif** menyala jika klasifikasi boleh digunakan.
9. Klik **Buat** untuk menyimpan.

## Mengubah Klasifikasi

1. Buka menu **Klasifikasi Surat**.
2. Pilih data klasifikasi yang ingin diperbarui.
3. Klik **Edit**.
4. Ubah data yang diperlukan.
5. Simpan perubahan.

## Menonaktifkan Klasifikasi

1. Buka data klasifikasi.
2. Klik **Edit**.
3. Matikan status **Aktif**.
4. Simpan perubahan.

## Catatan Penggunaan

- Menonaktifkan klasifikasi lebih aman daripada menghapus jika data pernah dipakai pada surat.
- Pastikan kode klasifikasi tidak duplikat.
- Klasifikasi yang rapi akan membantu nomor surat tetap konsisten.
