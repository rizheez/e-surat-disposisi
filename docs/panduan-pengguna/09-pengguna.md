# Panduan Menu Pengguna

## Fungsi Menu

Menu Pengguna digunakan untuk mengelola akun pengguna aplikasi, termasuk nama, email, jabatan, unit kerja, dan role.

## Hak Akses

Menu ini dikelola oleh admin.

## Menambah Pengguna

1. Buka menu **Pengguna**.
2. Klik **Buat**.
3. Isi **Nama**.
4. Isi **Email**.
5. Isi **Password**.
6. Isi **Jabatan** jika ada.
7. Pilih **Unit Kerja**.
8. Pilih **Role** sesuai tugas pengguna.
9. Klik **Buat** untuk menyimpan.

## Mengubah Pengguna

1. Buka menu **Pengguna**.
2. Pilih akun yang ingin diperbarui.
3. Klik **Edit**.
4. Ubah data yang diperlukan.
5. Jika password tidak ingin diubah, biarkan field password kosong.
6. Simpan perubahan.

## Mengatur Role Pengguna

1. Buka data pengguna.
2. Klik **Edit**.
3. Pilih role yang sesuai.
4. Simpan perubahan.
5. Minta pengguna logout dan login kembali jika perubahan akses belum terasa.

## Menghapus Pengguna

1. Buka menu **Pengguna**.
2. Pilih akun yang ingin dihapus.
3. Klik **Delete** jika tersedia.
4. Konfirmasi penghapusan.

## Catatan Penggunaan

- Gunakan email aktif dan unik untuk setiap pengguna.
- Pastikan unit kerja benar karena dipakai pada akses disposisi.
- Jangan memberikan role admin kepada pengguna yang tidak berwenang.
- Jika pegawai pindah unit, perbarui unit kerja akunnya agar alur disposisi tetap benar.
