# Panduan Menu Template Surat

## Fungsi Menu

Menu Template Surat digunakan untuk menyimpan format isi surat yang sering dipakai. Template membantu pengguna membuat isi surat lebih cepat dan lebih konsisten.

## Hak Akses

Menu ini dapat dikelola oleh admin atau staf administrasi sesuai hak akses sistem.

## Menambah Template Surat

1. Buka menu **Template Surat**.
2. Klik **Buat**.
3. Isi **Nama Template**.
4. Pilih **Unit Kerja** jika template hanya digunakan untuk unit tertentu.
5. Isi **Isi Template** menggunakan rich editor.
6. Klik **Buat** untuk menyimpan.

## Mengubah Template Surat

1. Buka menu **Template Surat**.
2. Pilih template yang ingin diperbarui.
3. Klik **Edit**.
4. Perbarui nama, unit kerja, atau isi template.
5. Simpan perubahan.

## Menghapus Template Surat

1. Buka menu **Template Surat**.
2. Pilih template yang tidak digunakan lagi.
3. Klik **Delete** jika tersedia.
4. Konfirmasi penghapusan.

## Tips Membuat Template

- Gunakan nama template yang jelas, misalnya "Surat Undangan Rapat" atau "Surat Pemberitahuan".
- Gunakan placeholder teks seperti "(Nama)", "(Tanggal)", atau "(Perihal)" jika isi perlu diganti pengguna.
- Hindari memasukkan data yang terlalu spesifik jika template akan dipakai berulang.
- Periksa hasil PDF jika template dipakai untuk surat keluar berbasis editor.
