# Panduan Menu Unit Kerja

## Fungsi Menu

Menu Unit Kerja digunakan untuk mengelola struktur unit atau bagian di dalam instansi. Data ini dipakai pada fitur pengguna, surat masuk, dan disposisi.

## Hak Akses

Menu ini dikelola oleh admin.

## Menambah Unit Kerja

1. Buka menu **Unit Kerja**.
2. Klik **Buat**.
3. Isi **Nama Unit Kerja**.
4. Isi **Kode** jika unit memiliki kode resmi.
5. Pilih **Parent Unit** jika unit tersebut berada di bawah unit lain.
6. Klik **Buat** untuk menyimpan.

## Mengubah Unit Kerja

1. Buka menu **Unit Kerja**.
2. Pilih data unit yang ingin diubah.
3. Klik **Edit**.
4. Perbarui data yang diperlukan.
5. Simpan perubahan.

## Menghapus Unit Kerja

1. Buka menu **Unit Kerja**.
2. Pilih unit yang ingin dihapus.
3. Klik **Delete** jika tersedia.
4. Konfirmasi penghapusan.

## Catatan Penggunaan

- Hindari menghapus unit kerja yang masih dipakai oleh pengguna atau disposisi aktif.
- Gunakan struktur parent unit jika instansi memiliki hierarki organisasi.
- Pastikan nama unit kerja ditulis konsisten agar mudah dicari.
