## Catatan Umum Akses

- Admin dapat mengelola seluruh data sesuai hak akses sistem.
- Pengguna non-admin hanya dapat melihat atau memproses data yang memang terkait dengannya, misalnya sebagai pembuat surat, penandatangan, penerima surat, atau penerima disposisi.
- Jika menu atau tombol tidak muncul, kemungkinan akun belum memiliki izin untuk aksi tersebut.
- Gunakan tombol notifikasi di bagian atas untuk melihat informasi baru dari sistem. Notifikasi tidak real-time, sehingga pengguna dapat refresh halaman jika ingin memastikan data terbaru tampil.
