# Panduan Menu Surat Masuk

## Fungsi Menu

Menu Surat Masuk digunakan untuk mencatat, melihat, mengelola, dan menindaklanjuti surat yang diterima oleh instansi. Dari menu ini, surat dapat dibuatkan disposisi, ditandai selesai, atau diarsipkan.

## Hak Akses

- Admin dan staf administrasi dapat melihat dan mengelola data surat masuk.
- Pengguna non-admin dapat melihat surat masuk jika dirinya menjadi penerima surat.
- Pengguna juga dapat melihat surat masuk jika menerima disposisi dari surat tersebut, baik langsung sebagai penerima user maupun melalui unit kerjanya.

## Membuat Surat Masuk

1. Buka menu **Surat Masuk**.
2. Klik tombol **Buat**.
3. Isi bagian **Informasi Surat**:
   - **Nomor Surat**: nomor surat dari pengirim.
   - **Nomor Agenda**: boleh dikosongkan jika ingin dibuat otomatis oleh sistem.
   - **Tanggal Surat**: tanggal yang tertulis pada surat.
   - **Tanggal Diterima**: tanggal surat diterima oleh instansi.
4. Isi bagian **Pengirim**:
   - **Nama Pengirim** wajib diisi.
   - **Alamat Pengirim** boleh diisi jika tersedia.
5. Isi bagian **Detail Surat**:
   - **Perihal** wajib diisi.
   - **Sifat Surat** pilih sesuai kebutuhan, misalnya biasa, penting, rahasia, atau segera.
   - **Prioritas** pilih rendah, sedang, tinggi, atau urgent.
   - **Penerima** dapat dipilih jika surat ditujukan kepada pengguna tertentu.
6. Isi bagian **Lampiran & Keterangan**:
   - Upload file surat jika tersedia.
   - File dapat berupa PDF atau gambar hasil scan.
   - Isi keterangan tambahan jika diperlukan.
7. Klik **Buat** untuk menyimpan.

## Membuat Disposisi dari Surat Masuk

1. Buka menu **Surat Masuk**.
2. Cari surat yang akan didisposisikan.
3. Klik aksi **Buat Disposisi**.
4. Pilih penerima disposisi, bisa user tertentu atau unit kerja.
5. Isi instruksi disposisi.
6. Tambahkan catatan dan batas waktu jika diperlukan.
7. Simpan disposisi.

## Menandai Surat Selesai

1. Buka data surat masuk.
2. Klik aksi **Tandai Selesai** jika pekerjaan terhadap surat tersebut sudah selesai.
3. Status surat akan berubah menjadi selesai.

## Mengarsipkan Surat Masuk

1. Buka menu **Surat Masuk**.
2. Pilih surat yang sudah tidak aktif diproses.
3. Klik aksi **Arsipkan**.
4. Surat akan pindah ke menu **Arsip Surat**.

## Catatan Penggunaan

- Pastikan nomor surat dan tanggal surat sesuai dengan dokumen asli.
- Gunakan disposisi jika surat perlu ditindaklanjuti oleh unit atau pengguna lain.
- Jangan menghapus surat yang masih memiliki proses disposisi aktif, kecuali benar-benar diperlukan dan sesuai kewenangan.
